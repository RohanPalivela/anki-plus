# Anki-Plus — Speedrun (MCAT)

[![Build Status](https://github.com/ankitects/anki/actions/workflows/ci.yml/badge.svg)](https://github.com/ankitects/anki/actions/workflows/ci.yml)
[![Documentation](https://img.shields.io/badge/docs-dev--docs.ankiweb.net-blue)](https://dev-docs.ankiweb.net)

**Exam: MCAT.** This is an AGPL-3.0 fork of [Anki](https://apps.ankiweb.net)
that turns it into a question-first study app for the **MCAT** (scale 472–528).
It adds a Rust engine change (question-gated card activation) and honest
Memory / Performance / Readiness predictions on top of Anki's spaced-repetition
core. It is **not affiliated with or endorsed by the AAMC**; MCAT is a
registered trademark of the AAMC.

This repo contains the source code for the computer version of
[Anki](https://apps.ankiweb.net).

## About

Anki is a spaced repetition program. Please see the [website](https://apps.ankiweb.net) to learn more.

## MCAT question bank & content credits

The Speedrun fork ships a vendored, **legally reusable** bank of MCAT-relevant
practice questions, imported once on desktop as native Anki notes (Tools →
_Speedrun (MCAT)_ → _Import question bank_). Because they are native objects,
one import **syncs to Android and every other device** through Anki's normal
sync — no per-device re-import. The bank is regenerated by
[`tools/speedrun/build_question_bank.py`](tools/speedrun/build_question_bank.py)
and stored (gzipped) at `pylib/anki/data/speedrun_question_bank.json.gz`.
